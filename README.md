# fastify-mvc

#### Structure Project
```
  - src
       |- controllers
       |- models
       |- app.ts
       |- router.ts
       |- server.ts
 - typings
```


#### NPM Install
```
$ npm i or npm install
```

#### Usage

```
--- command for dev ---
$ npm run dev
$ npx nodemon
```

```
--- command for build ---
$ npm run build
$ tsc -p tsconfig.json
```
